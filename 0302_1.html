8/21 Day 1｜抵達東京門戶

🏠 住宿區域：成田機場周邊

接駁資訊

成田國際機場 (NRT) ➔ 接駁車 (15m) ➔ 機場周邊飯店
20:00 抵達成田機場辦理入境與領取行李
21:30 飯店 Check-in搭乘飯店免費接駁巴士
22:00 宵夜時間飯店內便利商店最後補給
㊙️ 專業提示
第一晚建議住機場旁，明天一早回機場取車最方便，也能讓長輩早點休息。

8/22 Day 2｜花海與海中鳥居

🏠 住宿區域：鬼怒川溫泉

本日總車程：約 3h 15m (225km)

成田機場 ➔ 1h 10m ➔ 常陸海濱公園 ➔ 20m ➔ 那珂湊漁港 ➔ 1h 45m ➔ 鬼怒川
08:30 機場取車辦理租車手續，檢查車況
10:30 國營常陸海濱公園搭乘遊園小火車，漫步見晴之丘
13:00 那珂湊漁港午餐海味市場自由挑選生魚片、烤海鮮
15:00 大洗磯前神社欣賞太平洋巨浪沖刷神磯鳥居
17:30 抵達鬼怒川溫泉換浴衣享用和風晚宴
👨‍👩‍👧‍👦 親子/長輩友善
常陸海濱公園有**接駁火車**，小孩有大草坪放電，長輩免走路。
🍴 必吃
漁港現剖「特大生蠔」，一顆僅需約 500-800 日圓。

8/23 Day 3｜江戶傳奇與世界遺產

🏠 住宿區域：鬼怒川溫泉

本日總車程：約 1h 20m (短程輕鬆)

飯店 ➔ 15m ➔ 日光江戶村 ➔ 25m ➔ 日光東照宮 ➔ 30m ➔ 飯店
09:00 日光江戶村 (EDO WONDERLAND)忍者秀、變裝體驗，全家人互動性最高
13:30 日光東照宮看三猿、眠貓與陽明門的極致建築
16:00 鬼怒川遊船 / 纜車坐船順流而下或搭纜車眺望溪谷
👨‍👩‍👧‍👦 節奏建議
長輩若體力有限，東照宮可參觀重點即離開。江戶村的表演皆有座位，非常舒適。
㊙️ 秘境
東照宮旁的「二荒山神社」有求好運的巨大擲圈遊戲，小孩很喜歡。

8/24 Day 4｜尋找四萬藍與日本名湯

🏠 住宿區域：草津溫泉

本日總車程：約 3h 10m (145km)

鬼怒川 ➔ 2h ➔ 奧四萬湖 ➔ 15m ➔ 積善館 ➔ 55m ➔ 草津溫泉
08:00 早起出發今日山路較多，建議緩速慢行
10:30 奧四萬湖環湖自駕，欣賞鈷藍色的絕美湖水
12:30 四萬溫泉 積善館在神隱少女原型紅橋前拍照、午餐
15:30 草津溫泉 Check-in飯店稍作休息後換浴衣出門
19:00 湯畑夜景感受硫磺煙霧與華麗燈光秀
㊙️ 體驗
草津「熱之湯」的揉湯表演 (Yumomi)，是長輩最推薦的必看文化表演。
🍴 必吃
老街上的「溫泉饅頭」，配著免費的熱茶是最高享受。

8/25 Day 5｜國道最高點與栗子秘境

🏠 住宿區域：長野市區

本日總車程：約 2h 45m (景觀公路)

草津 ➔ 50m ➔ 國道最高點 ➔ 1h 10m ➔ 小布施 ➔ 45m ➔ 善光寺
09:00 志賀草津高原道路全日本海拔最高的景觀國道自駕
10:00 國道最高點 (2172m)下車拍照，視野可達富士山(天氣好時)
11:30 小布施秘境小鎮造訪岩松院北齋天頂畫、栗子小徑散步
15:30 長野 善光寺日本古剎巡禮，體驗地下戒壇摸鑰匙
🍴 必吃
小布施的「栗子飯」與「蒙布朗」。長野名物「信州蕎麥麵」。
👨‍👩‍👧‍👦 貼心提示
小布施路段平緩，非常適合推推車或長輩緩步。

8/26 Day 6｜神話森林與輕井澤

🏠 住宿區域：輕井澤

本日總車程：約 2h 20m (110km)

長野 ➔ 45m ➔ 戶隱奧社 ➔ 15m ➔ 鏡池 ➔ 1h 20m ➔ 輕井澤
08:30 出發前往戶隱避開中午人潮，森林芬多精最新鮮
09:30 戶隱神社奧社參道步行在 500 年巨杉下，吸收神聖能量
12:00 戶隱鏡池湖畔美景午餐，欣賞連峰倒影
16:00 抵達輕井澤入住森林別墅，呼吸清爽空氣
👨‍👩‍👧‍👦 體力建議
戶隱奧社單程步行需 30-40 分。長輩若體力不支，可在「隨神門」處休息折返，風景已足夠震撼。

8/27 Day 7｜輕井澤悠閒生活

🏠 住宿區域：輕井澤

本日總車程：約 1h (市區短程)

白絲瀑布 ➔ 15m ➔ 榆樹街小鎮 ➔ 10m ➔ Outlet
09:30 白絲瀑布 (Shiraito Falls)停車場步行 3 分鐘即達，極度推薦
11:30 榆樹街小鎮 / 石之教會溪邊午餐、丸山咖啡品味
14:30 繪本之森 / 玩具博物館小孩必訪，長輩可看精緻花園

🍴 必吃
中輕井澤的「川上庵」蕎麥麵或星野區的義大利麵。
